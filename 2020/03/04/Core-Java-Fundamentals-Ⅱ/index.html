使用命令行工具

  • javac命令启动Java编译器，它将.java文件编译成.class文件。
  • java命令启动Java虚拟机，虚拟机执行编译器放在class文件中的字节码。

注意：使用java命令的时候，编译的程序后面不用加.class后缀
